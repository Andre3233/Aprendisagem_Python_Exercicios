Este repositório reúne pequenos projetos e exercícios em Python que fui criando enquanto aprendo a linguagem. Cada script foca conceitos diferentes, como:
Estruturas de repetição (while, for);
Condicionais (if/else);
Validação de input do utilizador;
Contadores e acumuladores;
Manipulação de variáveis;
Tuplas;
Listas;
Dicionarios;
Funções;
Modularização;
E Tratamento de erros.
